# Assets

Branding and image assets for `solana-infra-doctor` — logo marks, wordmarks,
icons, favicon, and the GitHub social preview. **Not** shipped in the published
crate (this directory is listed under `exclude` in `Cargo.toml`).

See [`BRAND.md`](./BRAND.md) for the concept, color palette, and per-asset usage
rules. README/docs headers use `logo-wordmark-{light,dark}.svg`; the GitHub
avatar/crates.io mark uses `logo-mark-dark.svg`; the repo social preview uses
`github-social-preview.png`.
