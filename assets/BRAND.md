# solana-infra-doctor Brand Assets

## Primary concept
A validator/node network hexagon combined with a diagnostic heartbeat line.

## Brand names
- Project: `solana-infra-doctor`
- CLI binary: `sol-doctor`

## Colors
- Mint: `#34D399`
- Cyan: `#22D3EE`
- Violet: `#8B5CF6`
- Dark background: `#080B10`
- Light text: `#F3F4F6`
- Muted text: `#94A3B8`

## Usage
- Use `logo-mark.svg` for transparent backgrounds.
- Use `logo-mark-dark.svg` for GitHub avatar, crates.io, and dark UI.
- Use `logo-wordmark-dark.svg` for README and documentation headers.
- Use `favicon.svg` or `icon-32.png` for website favicon.
- Use `github-social-preview.png` as the GitHub social preview image.

## Restrictions
- Do not recolor the logo with Solana's exact official gradient.
- Do not place the mark on visually noisy backgrounds.
- Do not add medical crosses, cartoon doctor imagery, or extra text inside the mark.
